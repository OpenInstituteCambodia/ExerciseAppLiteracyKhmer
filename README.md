# SAGE Ionic2 Application Sample
---

## Getting Started

1. `git clone https://github.com/socheatsok78/SAGE-Ionic2-Application-Sample.git`
2. `cd SAGE-Ionic2-Application-Sample`
3. `ionic state restore`
4. `npm install`
5. `bower install`
6. Done
